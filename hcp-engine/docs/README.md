# hcp-engine/docs — historical PhysX-era reference

> **Superseded — historical.** This directory holds PhysX-era consultation and reference material
> (`physx5-reference.md`, `physx-math-reference.md`, `physx-superposition-*`, `physx-phase2-*`,
> `o3de-physx-wrapping.md`, etc.) from when the settle was a PhysX-PBD particle simulation. **PhysX
> has been fully removed** and the engine is now AZSL/host-resident. These files are kept as a record
> of the design path that led here; they do **not** describe the current engine.
>
> The current GPU/engine reference lives in [`docs/azsl-training-corpus/`](../../docs/azsl-training-corpus/).
